# ⚽ Sahur Ball

A high-energy, chaotic 2-player soccer game with dynamic physics, crazy power-ups, and intense ball-bumping action!

## 🎮 How to Play

### Controls

**Player 1:**
- **A / D** — Move left / right
- **W** — Jump

**Player 2:**
- **← / →** — Move left / right
- **↑** — Jump

**Mobile:** Touch buttons appear at the bottom of the screen

### Objective

Score 5 points to win the match! Bump the ball into your opponent's goal while dealing with random chaos effects that shake up the game every 3 seconds.

### Game Features

- **Dynamic Physics:** Realistic ball physics with spin, collision detection, and momentum
- **Chaos Effects:** Random game-changing effects like:
  - 🔄 Reverse controls
  - 🌙 Low gravity / Super gravity
  - 🌪️ Wind gusts
  - 🪙 Ice floors
  - 🔥 Speed boosts
  - 🎆 Disco visuals
  - 🌍 Earthquakes
  - And many more!

- **Streaks & Fire Mode:** Build consecutive wins to trigger "on fire" mode
- **Power-ups:** Collect power-ups for temporary advantages
- **Responsive Design:** Works on desktop, tablet, and mobile devices

## 🚀 Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Play!

No build process or dependencies required — it's pure HTML5 Canvas + JavaScript.
